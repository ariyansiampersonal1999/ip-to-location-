# 📡 IP TO LOCATION 
A simple and powerful Python-based IP lookup tool that provides detailed information about any IP address.
🚀 Features
🔍 Check your own public IP
🌐 Lookup any target IP
📍 Get location details (Country, City, Region)
🛰 ISP & Organization info
🕒 Timezone details
🗺 Direct Google Maps location link
⚡ Fast & lightweight CLI tool

###
⚠️ Note
This tool uses a free API → ip-api.com
API may have rate limits
Internet connection required
